# Project Coding Guidelines

## Naming

- **Public API**: `snake_case`
- **Internal code**: `snake_case`
- **Type parameters**: `T_prefix_snake_case` (e.g., `T_data`, `T_transformer`)
- **Loop/reduce variables**: `acc`, `k`, `v`, `_` are acceptable

## Patterns

**Map/Object lookups**: Store result in variable, don't fetch twice

```typescript
// ✅ Good
const tag_map = db.get(tag);
if (tag_map) {
  tag_map.delete(name);
}

// ❌ Bad
if (db.get(tag)) {
  db.get(tag)!.delete(name);
}
```
